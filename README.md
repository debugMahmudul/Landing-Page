# Landing-Page

This is a non-responsive landing page. It demonstrates foundational HTML and CSS skills, including layout techniques with Flexbox, responsive design, and semantic markup.

## Built With

- HTML5
- CSS
- Flexbox

## Features

- Responsive layout
- Centered quote and testimonial
- Call-to-action section with styled button
- Clean, minimal footer
